## Who I am

I'm an **AI-first developer** — I think in products, then use AI to build them fast and ship them for real. I define *what* needs to exist and *why*, lean hard on tools like [Claude Code](https://claude.ai/code) for the *how*, and stay in the loop on architecture, review, and iteration until it's right.

The result: working software, in production, not tutorial projects. Everything below is **live** — click and use it.

> *Don't just learn to code. Build things people can actually use.*
